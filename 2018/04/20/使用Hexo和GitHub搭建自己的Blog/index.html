使用 Hexo 和 GitHub 搭建自己的 Blog

Posted on 2018-04-20 | In Life

简单地将这个过程分为 3 步骤:

  1. 在 GitHub 上配置 GitHub Pages
  2. 本地使用 Hexo 搭建自己的博客
  3. 将 Hexo 上传至 GitHub

配置 GitHub Pages

在 GitHub 上配置 GitHub Pages 很简单，进入 自己同名 的 Repositories 中点击设置 Setting，下拉找到 GitHub Pages 设置 Choose a theme 一个主题，就成功开启了 GitHub Pages。
可以在这个网址下 https://"你的GitHub用户名".github.io/ 看到设置效果。
